E N G L I S H

:::Description:::
This is a simple script that allows to search a Keyword in the metatags of some files.
The script makes a parsing to all files of the selected directory (View settings.php), to find the Description, the title, and the keywords of each page.
There are three example files (Page1.htm, Page2.php, Page3.PHP3), view the Html source of these files.
This is the structure:

<HTML>
<HEAD>
<META Name="description" Content="Description of the page">
<META Name="keywords" Content="Keyword1 keyword2 keyword3 keywordX">
<TITLE>Title of the page</TITLE>
</head>

Html of the page...

</html>

In the 3rd row of each page you must insert the Description;
in the 4th row there are the keywords, separated by spaces;
in the 5th row there is the title.

Warning!!! you must insert each Meta-tag in a different line:
<Meta Name=... Content=...><Meta Name=... Content=...><title>....this makes an error!!!

You can exclude some files from parsing by leaving the Meta-Keywords Box blank.
Example:
...
<META Name="description" Content="Description of the page">
<META Name="keywords" Content="">
<TITLE>Title of the page</TITLE>
...

:::Usage:::
First Unzip all files and then upload them to the same directory.
Next edit the settings.php file to customize the scripts.
Next launch the index.php script (http://www.yourspace.com/dir-where-the-files-were-unzipped/index.php) and enjoy it!
You can test the search engine by inserting the 'news' or 'reviews' keywords.
Your webspace must support the Php scripting.

:::Descrizione:::
Questo è un semplice script che cerca una parola chiave nei meta-tags di alcuni files.
Lo script esegue un parsing in tutti i files della directory selezionata (Vedi settings.php), per trovare la Descrizione, il titolo, e le parole-chiave associate ad ogni pagina.
Ci sono tre files di esempio (Page1.htm, Page2.php, Page3.PHP3), vedi il sorgente Html di questi.
Questa è la struttura:

<HTML>
<HEAD>
<META Name="description" Content="Descrizione della pagina">
<META Name="keywords" Content="Chiave1 chiave2 chiave3 chiaveX">
<TITLE>Titolo della pagina</TITLE>
</head>

Html della pagina...

</html>

Nella terza riga di ogni pagina devi inserire la Descrizione;
Nella quarta riga ci sono le parole-chiave, separate da spazi;
Nella quinta riga c'è il titolo.

Attenzione!!! è necessario che ogni Meta-tag venga inserito in una riga differente:
<Meta Name=... Content=...><Meta Name=... Content=...><title>....questo provocherebbe un errore!!!

Puoi escludere alcuni files dal parsing lasciando vuoto il campo delle Meta-Keywords.
Esempio:
...
<META Name="description" Content="Descrizione della pagina">
<META Name="keywords" Content="">
<TITLE>Titolo della pagina</TITLE>
...
