E N G L I S H

:::Description:::
This is a script that generates a counter in a page.
Install the script and launch yourpage.php for more details.
You can select one of the 3 visualization modes: Text-mode, Graphical-mode, Invisible-mode.

[More Details On-Line >>]

:::Usage:::
First Unzip all files and then upload them to the same directory.
Next launch the yourpage.php script (http://www.yourspace.com/dir-where-the-files-were-unzipped/yourpage.php) and enjoy it!
Your webspace must support the Php scripting.
