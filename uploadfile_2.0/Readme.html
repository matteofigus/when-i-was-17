E N G L I S H

:::Description:::
This is a script that allows to upload some files to remote server from local machine.
The upload process starts opening the script with a simple browser that support the file upload, selecting the files by clicking on the Browse... button and finally clicking on the UPLOAD button.
The files will be saved in a directory that the script provides to create the first time you launch the script.
[More Details On-Line >>]

:::Usage:::
First Unzip all files and then upload them to the same directory.
Next edit the settings.php file to customize the scripts.
Next launch the index.php script (http://www.yourspace.com/dir-where-the-files-were-unzipped/index.php) and enjoy it!
Your webspace must support the Php scripting.

:::New Features From Version 1.0:::
The script is protected by a password.

:::Descrizione:::
Questo è uno script per l'upload di files in un server remoto dalla propria macchina.
L'invio dei files avviene aprendo lo script con un semplice browser che supporti l'upload di files, selezionando i files facendo click sul pulsante Sfoglia... e facendo in fine click su UPLOAD.
I files saranno salvati in una directory che lo script provvederà a creare la prima volta che lo script verrà lanciato.
[Maggiori Dettagli On-Line >>]

:::Utilizzo:::
Prima di tutto decomprimere tutti i files ed uploadarli nel proprio spazio web nella stessa directory.

Dopo lanciate lo script index.php (http://www.tuospazioweb.com/dir-dove-stanno-i-files/index.php) e buon divertimento!
